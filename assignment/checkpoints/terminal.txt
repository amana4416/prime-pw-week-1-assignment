# 8 - Why would you run the command 'git status'?
You would run 'git status' see if your your files are in the staging area. If they are shown to be in the staging area, your files(s) will be shown in green colored text.

# 9 - What does the command 'git add .' do?
'git add .' allows you to add all of your code changes to the staging area. You want your work added to the staging area because it is like a waiting room for your code changes to show they are ready to be saved (pushed) to GitHub.
